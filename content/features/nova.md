## AI-Powered Creation

Bilup Nova brings the power of AI to your Scratch projects. Use your favorite AI assistant to help you create, modify, and optimize your projects with unprecedented ease and creativity.

### AI Integration

#### Multi-Provider Support
- OpenAI GPT integration
- Claude AI support
- Custom API key configuration
- Flexible provider switching

### Block Editing

#### AI-Assisted Creation
- Create new blocks with AI help
- Modify existing blocks intelligently
- Optimize block logic
- Debug and fix issues

#### Natural Language Commands
- Describe what you want to create
- AI translates your ideas into blocks
- Iterative refinement through conversation

### Visual Creation

#### Simple SVG Appearances
- Create simple SVG sprite appearances
- Basic shapes and designs
- Suitable for simple projects

### Customization

#### API Configuration
- Add your own API key
- Choose your preferred provider
- Adjust AI parameters
- Control response style

#### Privacy & Control
- Your API key stays private
- Full control over AI usage
- Optional AI assistance
- Manual override always available

Bilup Nova is designed to enhance your creative workflow — giving you AI-powered tools while keeping you in full control of your projects.

## Feature for Teams

Work together on Scratch projects in real time. Our Live Collaboration feature enables multiple users to edit scripts, sprites, and sounds simultaneously — making teamwork seamless and intuitive.

### Real-Time Editing

#### Multi-User Support
- Multiple users editing at the same time
- Live updates across all connected clients
- Synchronized scripts, sprites, and sounds
- Smooth conflict handling

#### Project-Wide Collaboration
- Edit blocks collaboratively
- Modify costumes together
- Manage and adjust sounds in real time
- Shared access to the full project workspace

### Communication

#### Built-In Comments
- Add comments directly to the project
- Discuss ideas without leaving the editor
- Coordinate changes efficiently
- Keep collaboration organized and contextual

### Access & Sharing

#### Instant Join
- No registration required
- Join instantly via shared link
- Quick session setup
- Frictionless onboarding for new collaborators

#### Privacy Controls
- Public collaboration sessions
- Private mode with join approval
- Host-controlled access management
- Only approved users can enter private sessions

Our Live Collaboration system is designed to make teamwork effortless — whether you're teaching, learning, or building complex projects together in real time.
